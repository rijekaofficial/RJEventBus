# RJEventBus
Extremely fast event system for Kotlin.

## Gradle
```groovy
repositories {
    maven("https://rijekaofficial.github.io/maven-repo")
}

dependencies {
    implementation("dev.rijeka.rjeventbus:rjeventbus:1.0.0")
}
```
